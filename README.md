# Sign-Language-Recognition
Real-time gesture detection

---
## 📌 Project Overview

This project is designed to recognize sign language gestures in real time using Computer Vision techniques. It utilizes MediaPipe Hands to detect 21 hand landmarks from webcam input and processes them through a FastAPI backend to identify gestures based on geometric hand features. The recognized gesture and confidence score are displayed instantly through a React-based user interface.
## 🚀 Features

- Real-time hand gesture recognition
- Live webcam-based detection
- Recognition of 15 different sign language gestures
- Confidence score prediction
- Gesture history tracking
- Interactive gesture guide
- Responsive user interface
- Fast REST API backend

## ⚙️ Workflow

1. Capture live webcam video
2. Detect 21 hand landmarks using MediaPipe
3. Send landmark coordinates to the backend
4. Process landmarks using the gesture recognition engine
5. Predict the corresponding sign
6. Display gesture, confidence score, and history
